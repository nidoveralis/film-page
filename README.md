# Film-page

## Описание

На главной странице приложения представлен список фильмов, доступных для просмотра.
Пользователи могут просматривать названия фильмов, описания, рейтинги, жанры и другие основные характеристики.
Каждый фильм в списке обычно представлен кратким описанием и основной информацией.

После выбора фильма из списка пользователи переходят на отдельную страницу с подробной информацией о выбранном фильме.
На этой странице могут быть раскрыты подробности сюжета, режиссера, актерского состава, рейтинги, отзывы, трейлеры и другая важная информация о фильме.
Пользователи могут оценивать фильм, оставлять комментарии, добавлять в свой список просмотров и делиться информацией с друзьями.

На странице конкретного фильма могут быть приведены рекомендации похожих фильмов.
Эта функция помогает пользователям находить другие фильмы, которые могут заинтересовать их на основе их предпочтений.
Подобные предложения могут быть предложены на основе жанра, рейтинга, актеров фильмов или других факторов, которые связаны с выбранным фильмом.
Это приложение по поиску фильмов предлагает пользователям удобный способ находить, изучать и просматривать широкий выбор фильмов, обогащая их популярный опыт.

## Запуск проекта

1. Клонируйте репозиторий
2. Установите необходимые зависимости ```npm i```
3. Запустите проект ```npm run start```
